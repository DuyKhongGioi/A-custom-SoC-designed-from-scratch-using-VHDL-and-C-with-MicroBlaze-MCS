# A-custom-SoC-designed-from-scratch-using-VHDL-and-C-with-MicroBlaze-MCS
A custom SoC designed from scratch featuring self-developed IP cores (Timer, GPIO, UART, I2C, SPI) for sensor and PC interfacing. Based on AMD's MicroBlaze soft-core, utilizing VHDL for hardware description and C++ for software application
